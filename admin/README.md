# VPN Startup — Admin Panel (Next.js)

Веб-панель для управления VPN-сервисом.

## Функции

- **Серверы**: просмотр списка (GET /api/servers). Добавление/редактирование — через БД или отдельные admin API (требуют реализации в backend).
- **Пользователи**: список, блокировка (требует admin API в backend).
- **Подписки**: просмотр и управление (admin API).
- **Статистика трафика**: дашборд (при наличии traffic_logs и API).

## Запуск

```bash
pnpm install
pnpm dev
```

Откройте http://localhost:3001. Для доступа к защищённым admin API нужна авторизация (JWT с ролью admin) — реализуется в backend.
